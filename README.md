# NukeJS Beta 1.0.1

On this version our only work in web platform, but it will be useful great than. Also easy to implement in Laravel, Spring Boot ... 

## How to install? 

Go to root project and run 
    `npm install`

## Development Server

Go to root project and run 
    `npm run dev`

## How to build

Go to root project and run 
    `npm run build`
